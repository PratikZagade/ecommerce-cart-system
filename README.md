<h2>📌 Overview</h2>
<p>
A modern e-commerce web application that allows users to browse products, add items to a cart, update quantities, and complete checkout with dynamic pricing.
</p>

<h2>🚀 Features</h2>
<ul>
<li>🛍️ Product Listing</li>
<li>➕ Add to Cart</li>
<li>➖ Remove Items</li>
<li>🔄 Update Quantity</li>
<li>💰 Real-time Price Calculation</li>
<li>🧾 Checkout Page</li>
<li>📱 Responsive Design</li>
</ul>

<h2>⚙️ Tech Stack</h2>
<p>
HTML, CSS, JavaScript, React.js, Redux, React Router
