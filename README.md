    <h2>📌 Project Overview</h2>
    <p>
        The E-Commerce Cart System is a modern web-based application designed to simulate an online shopping experience. 
        It allows users to browse products, add items to a cart, update quantities, and proceed to checkout.
    </p>

    <p>
        A shopping cart system acts as the core component of any e-commerce platform, enabling users to manage selected 
        items and complete transactions smoothly. :contentReference[oaicite:0]{index=0}
    </p>

    <h2>🚀 Features</h2>
    <ul>
        <li>🛍️ Product Listing with Details</li>
        <li>➕ Add to Cart Functionality</li>
        <li>➖ Remove Items from Cart</li>
        <li>🔄 Update Product Quantity</li>
        <li>💰 Dynamic Price Calculation</li>
        <li>🧾 Checkout Page</li>

    <h2>🧠 How It Works</h2>
    <p>
        The system allows users to browse products and add them to a cart. The cart maintains selected items and 
        calculates the total price dynamically. Users can modify quantities or remove items before proceeding to checkout.
    </p>

        Shopping cart systems manage product selection, pricing, and checkout operations, acting as the backbone of
